# mixedimageextra
Надстройка для mixedImage дополнительной конфигурации для управления генерацией картинок

В настройках TV mixedImage в поле Возможные варианты можно указать дополнительные настройки в виде JSON

{
  "imageConfig": {
    "quality": 85,
    "sizes": [
      {"suffix": "", "w": 0, "h": 0},
      {"suffix": "-L", "w": 645, "h": 373},
      {"suffix": "-M", "w": 280, "h": 162},
      {"suffix": "-S", "w": 400, "h": 232}
    ]
  }
}

Плагин сохраняет любой формат в jpg и создает копии согласно настройкам json добавляя суффиксы к имени оригинального файла

Обрезка и сжатие с помощью php модуля Imagick

Ориентация вычисляется автоматически и если вертикальное фото то вместо кропа делает ресайз по высоте.

Плагин сохраняет все EXIF метаданные

Работает с MIGX - названиям добавляется через дефис id поля migx начиная со второго элемента.

При удалении удаляется сразу фото со всеми копиями ресайзами.

=============

Для работы нужно переопределить событие загрузки фото. Для этого 

1) создаем плагин MixedImageExtra 
2) создадим структуру папок (в репозитории все есть)

==============

ДОПОЛЬНИТЕЛЬНО:

Также присутсвтвует сниппет который расширяет возможности генерации пути сохранения картинок. Задается в поле Путь сохранения например так 
@SNIPPET images.parent.category - разделил точками для наглядности.

Затем нужно создать этот сниппет images.parent.category - пример есть в репозитории
